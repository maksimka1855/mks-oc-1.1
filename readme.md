# MKS‑OC 1.1

Hi! This is the official repository for MKS‑OC 1.1 — a custom operating system for virtual worlds.  
The **Start menu panel** is already operational!

## Project Structure

The repository includes:

- `mks-oc-1.1.exe` — a ready‑to‑use executable (double‑click to launch).  
- `/source/` — full Python source code (see details below).

## How to Launch

1. **Download the repository**:  
   - Click the *«Code»* button on GitHub → *«Download ZIP»*.  
   - Or use `git clone` if Git is installed:  
     ```bash
     git clone https://github.com/your-username/mks-oc.git
     ```

2. **Extract the archive** (if downloaded as ZIP).  
3. **Locate the `.exe` file** in the root folder.  
4. **Launch the application**: Double‑click `mks-oc-1.1.exe`.
