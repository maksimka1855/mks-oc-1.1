# MKS‑OC 1.1

Hi! This is the official repository for MKS‑OC 1.1. Below are instructions on how to run the project.

## Project Structure

The repository contains:
- The executable `.exe` file (ready‑to‑use version),
- The `source/` folder (contains the full Python source code).

## How to Run

1. **Download the repository**:  
   - Click the «Code» button on GitHub and select «Download ZIP».
   - Or use `git clone` if you have Git installed.

2. **Unzip the archive** (if downloaded as ZIP).

3. **Locate the `.exe` file** in the root folder.

4. **Launch the app**: Double‑click the `.exe` file to start MKS‑OC 1.1.

## Important Notes

- **Administrator privileges**: You do *not* need full admin rights to run the app.
- **Safety**: The application is designed to be safe for regular use. Always ensure you have backups of important data.

## Development Details

- **IDE**: The project was developed in **PyCharm**.
- **Source code**: All Python files are in the `source/` folder. You can:
  - Study the code,
  - Contribute improvements (via pull requests),
  - Adapt the project for your needs.
